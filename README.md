# Eventime server

## Getting Started

### Prerequisites

Make sure you have the following installed:

- Node.js
- npm
- Docker

### Installation

1. **Clone the repository**:

   ```sh
   git clone <repository-url>
   cd eventime-server
   ```

2. **Install Dependencies**:
   ```sh
   npm install
   ```

### Starting the Server

1. **Start Docker Containers**:

   ```sh
   docker-compose up
   ```

2. **Start the Server**:
   ```sh
   npm run start
   ```

### Building the Server

1. **Build the Project**:
   ```sh
   npm run build
   ```

### Running Tests

1. **Run Tests**:
   ```sh
   npm test
   ```
